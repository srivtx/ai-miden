# Phase 100: Automated Circuit Discovery (MechInterp) — Summary

This phase introduced the field of mechanistic interpretability and tools for understanding internal model computation.

## Key Concepts

- **Mechanistic Interpretability:** Reverse-engineering neural networks to find interpretable circuits that implement specific behaviors.
- **Attribution Patching:** A gradient-based method to attribute the effect of an intervention to specific layers or components.
- **Sparse Autoencoder (SAE):** A model trained to disentangle superposed activations into sparse, interpretable features.

## Takeaway
Understanding what models do is as important as making them bigger. Tools like attribution patching and sparse autoencoders give us a path from black-box weights to human-understandable algorithms, enabling safer and more reliable AI systems.
