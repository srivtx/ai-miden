# Phase 92 Summary: Benchmark Design & Evaluation Science

This phase explored how to design evaluations that measure true capability rather than memorization.

- **Benchmark**: A standardized dataset plus protocol that enables fair comparison across models.
- **Contamination**: Train-test leakage that inflates scores and destroys generalization claims.
- **Validity**: Whether the benchmark measures the construct it claims to measure.

Good evaluation science requires clean splits, human baselines, and skepticism about headline numbers.
