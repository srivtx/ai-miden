# Phase 94 Summary: Experiment Design & Statistical Rigor

This phase introduced statistical tools for trustworthy ML evaluation.

- **Confidence Interval**: A range that captures the true metric with a known probability.
- **p-value**: A measure of how surprising the observed data would be if there were no real effect.
- **Variance Reduction**: Techniques like cross-validation and averaging that stabilize noisy estimates.

Using these tools prevents false discoveries and ensures that reported improvements are real.
