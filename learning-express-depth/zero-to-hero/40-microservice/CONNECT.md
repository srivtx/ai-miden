## The Foundation

Every project in this path introduced a *concept* and a *pattern*:

- **HTTP** (projects 1-6): requests, responses, routing, query strings, body parsing, cookies
- **Identity** (projects 7-12): Express, bcrypt, JWT, SQLite, foreign keys, migrations
- **Robustness** (projects 13-18): ORM, validation, error handling, logging, REST, pagination
- **Operations** (projects 19-27): search, upload, email, cache, Redis, rate limit, cron, queue, transactions
- **Real-time** (projects 28-32): WebSocket, SSE, presence, CRDT, WebRTC
- **Production** (projects 33-40): RBAC, webhooks, payments, tests, Docker, CI/CD, observability, microservices

These patterns apply to any backend. Node, Python, Go, Java, Rust — the patterns are the same.
